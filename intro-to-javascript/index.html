What Is JavaScript?


  • A programming language used (primarily) for
    client-side web development:
    • DOM Manipulation
    • User Interaction
    • Asynchronous Communication
    • ... And Much More!

A Brief History

Source: Wikipedia


  • First released in 1995 for Netscape Navigator 2.0
  • Implemented by Microsoft for IE3 in 1996 as "JScript"
  • Language standardized as "ECMAScript" in 1997
  • Hated by developers ever since*


Tidbit: The first server-side implementation of JavaScript was released in '94 for Netscape Enterprise Server

Using JavaScript

(In the context of the broswer)

  • In a separate file included in an HTML document
    
<script type="text/javascript" src="script.js"></script>
  • In a <script> block in an HTML document
    
<script type="text/javascript">
  console.log("Hello, World!");
</script>
  • In an attribute of a DOM element 
    
<button onclick="alert('Hello, World!');">Click Me</button>

Variables & Types


  • JavaScript is dynamically typed
  • All variables declared with var keyword
  • "Primitive" types
    • Numbers var x = 10;
    Strings var x = "Hello";
    Booleans var x = true;
    Null var x = null;
    Undefined

Variables & Types


  • "Everything is an object" in JavaScript
  • Additional Object Types
    • Arrays var x = [];
    Objects var x = {key:"value"};
    Regex var x = /pattern/;
    Functions var x = function() {};
    Maths, Dates, and more!

Control Flow


  • if statements
    
if (x === "Hello") {
  console.log("Greetings!");
}
else if (x === "Goodbye") {
  console.log("See ya later!");
}
else {
  	console.log("What was that?");
}
  • switch statements
    
switch(x) {
  case 0:
  case 1:
    console.log("Fallthrough!");
    break;
  case 2:
    console.log("Don't forget to break!");
    break;
  default:
   console.log("Default!");
}

Control Flow


  • while loop
    
var i = 0;
while (i < 10) {
  console.log(i);
  i++;
}
  • for loop
    
for (var i = 0; i < 10; i++) {
  console.log(i);
}
  • for in loop (for Objects)
    
for (var key in obj) {
  // Avoid iterating over prototype properties
  if (obj.hasOwnProperty(key)) {
    console.log(key);
  }
}
  • for each loop (for Arrays)
    
var arr = ["A", "B", "C"];
arr.forEach(function(value, index, array) {
  console.log(value);
});

Comparators


  • "Falsy" values: 0, false, undefined
  • "Truthy" values: everything else
  • Use "==" or "!=" to compare value
    
if (x == true) { console.log("Truthy!"); } // Shorthand: if (x) {}
if (x != true) { console.log("Falsy!"); } // Shorthand: if (!x) {}
  • Use "===" or "!==" to compare value AND type
    
if (typeof x === "undefined") { console.log("Undefined!"); }
if (x !== "Hello") { console.log("Strings Don't Match!"); }
