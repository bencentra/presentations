    • DOM Manipulation
    • User Interaction (Event-driven)
    • Asynchronous Communication (AJAX)
    • ... And Much More!

A Brief History

Source: Wikipedia


  • First released in 1995 for Netscape Navigator 2.0
  • Implemented by Microsoft for IE3 in 1996 as "JScript"
  • Language standardized as "ECMAScript" in 1997
  • Hated by developers ever since*


Tidbit: The first server-side implementation of JavaScript was released in '94 for Netscape Enterprise Server

Using JavaScript

(In the context of the broswer)

  • In a separate file included in an HTML document
    
<script type="text/javascript" src="script.js"></script>
  • In a <script> block in an HTML document
    
<script type="text/javascript">
	console.log("Hello, World!");
</script>
  • In an attribute of a DOM element 
    
<button onclick="alert('Hello, World!');">Click Me</button>

Variables & Types


  • JavaScript is dynamically typed
  • All variables declared with var keyword
  • "Primitive" types
    • Numbers var x = 10;
    Strings var x = "Hello";
    Booleans var x = true;
    Null var x = null;
    Undefined

Variables & Types


  • "Everything is an object" in JavaScript
  • Additional Object Types
    • Arrays var x = [];
    Objects var x = {key:"value"};
    Regex var x = /pattern/;
    Functions var x = function() {};
    Maths, Dates, and more!

Operators


  • Arithmetic
    
10 + 5; // 15
10 - 5; // 5
10 * 5; // 50
10 / 5; // 2
10 % 5; // 0
10++; // 11
10--; // 9
"10" + "5"; //"105"
  • Assignment
    
var x = 5; // 5
x += 5; // 10
x -= 5; // 5
x *= 5; // 25
x /= 5; // 5
x %= 5; // 0
  • Comparison
    
5 == 5; // true
5 === "5"; // false
5 != 10; // true
5 !== "5"; // true
5 > 3; // true
5 < 10; // true
5 >= 5; // true
5 <= 5; // true
  • "Ternary" Operator
    
(x > 5 ? console.log("more") : console.log("less"));

Control Flow


  • if statements
    
if (x === "Hello") {
  console.log("Greetings!");
}
else if (x === "Goodbye") {
  console.log("See ya later!");
}
else {
  	console.log("What was that?");
}
  • switch statements
    
switch(x) {
  case 0:
  case 1:
    console.log("Fallthrough!");
    break;
  case 2:
    console.log("Don't forget to break!");
    break;
  default:
   console.log("Default!");
}

Control Flow


  • while loop
    
var i = 0;
while (i < 10) {
  console.log(i);
  i++;
}
  • for loop
    
for (var i = 0; i < 10; i++) {
  console.log(i);
}
  • for in loop (for Objects)
    
for (var key in obj) {
  // Avoid iterating over prototype properties
  if (obj.hasOwnProperty(key)) {
    console.log(key);
  }
}
  • for each loop (for Arrays)
    
var arr = ["A", "B", "C"];
arr.forEach(function(value, index, array) {
  console.log(value);
});

Truthiness


  • "Falsy": 0, false, [], undefined, null, NaN
  • "Truthy": everything else
  • Use "==" or "!=" to compare value
    
if (x == true) { console.log("Truthy!"); } // Shorthand: if (x) {}
if (x != true) { console.log("Falsy!"); } // Shorthand: if (!x) {}
if (23 == "23") { console.log("WHAT?!"); } // true...!!!
  • Use "===" or "!==" to compare value AND type
    
if (typeof x === "undefined") { console.log("Undefined!"); }
if (x !== "Hello") { console.log("Strings Don't Match!"); }
if (23 === "23") { console.log("WHAT?!"); } // false

Using Arrays


  • Arrays are "high-level, list-like objects."
  • Arrays can contain elements of any type!
    
// Instantiate the array
var arr = new Array("One", "Two", "Three"); // Array Constructor
var arr = ["One", "Two", "Three"]; // Array Literal

// Common array operations
var length = arr.length; // Get array length 
var first = arr[0]; // Access elements by index
console.log(arr); // Print array

Using Arrays


  • Adding elements
    
var arr = ["One", "Two", "Three"];
arr.push("Four"); // End
arr.unshift("Zero"); // Beginning
arr.splice(1, 0, "One Half"); // Anywhere
  • Removing Elements
    
var arr = ["One", "Two", "Three"];
arr.pop(); // End
arr.shift(); // Beginning
arr.splice(1, 1); // Anywhere
  • Other useful stuff
    
var arr = ["One", "Two", "Three"];
arr.reverse(); // ["Three", "Two", "One"]
arr.sort(); // ["One", "Three", "Two"]
arr.sort(function(a,b) {
  if (a < b) return -1;
  else if (a > b) return 1;
  else return 0;
});
arr.join(); // "One,Two,Three"
arr.concat(["Eight", "Nine", "Ten"]);
arr.indexOf("Three"); // -1 if not found

Using Objects


  • Objects are key/value pairs.
  • Keys are typically strings, Values can be any type!
  • Example 1
    
// Create a new Car object
var car = new Object();

// Assign some "properties"
car["make"] = "Tesla"; 
car.model = "Roadster"; 
car.year = 2014; 

// Create a "method"
car.drive = function() {
  console.log("Driving my " + 
    this.make + " " + this["model"]);
}
  • Example 2
    
// Create a new Car object using a Literal
var car =  {
  make: "Tesla",
  model: "Roadster",
  year: 2014,
  drive: function() {
    console.log("Driving my " + 
      this.make + " " + this["model"]);
  }
};

Using Functions


  • Functions are first-class; can be passed like other objects
  • Unlike other objects, functions can be invoked.
  • Many different ways to declare and invoke functions:
    
function doSomething() { ... } // Named function
doSometing();

var doSomething = function() { ... } // Also a named function
doSomething();

object.doSomething = doSomething; // Assign as a method
object.doSomething();

// Anonymous function, passed as a parameter to another function!
button.addEventListener("click", function() { ... });
